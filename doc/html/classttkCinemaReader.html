Detailed Description

TTK VTK-filter that reads a Cinema Spec D Database.

Online examples:

  • Cinema IO example
Author
Jonas Lukasczyk jl@jl.nosp@m.uk.d.nosp@m.e
Date
01.09.2018

This filter can be used as any other VTK filter (for instance, by using the sequence of calls SetInputData(), Update(), GetOutput()).

Parameters
Outputcontent of the data.csv file of the database in form of a vtkTable

Online examples:

  • Merge Tree Clustering example
  • Merge Tree Temporal Reduction
