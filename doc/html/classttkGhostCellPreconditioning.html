Detailed Description

TTK VTK-filter to generate rank ownership information.

Author
Michael Will mswil.nosp@m.l@rh.nosp@m.rk.un.nosp@m.i-kl.nosp@m..de
Date
2022

This VTK filter generates a rankArray based on a global id array and ghost cell information.

Parameters
InputvtkDataSet with global ids and ghost cell information.
OutputvtkDataSet.
