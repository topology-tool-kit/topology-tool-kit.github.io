Detailed Description

TTK VTK-filter that requests data as long as it is available.

Author
Jonas Lukasczyk jl@jl.nosp@m.uk.d.nosp@m.e
Date
01.11.2018

This filter requests more data as long as the maximum number of elements is not reached. This filter works in conjunction with the ttkForEachRow filter.

Parameters
InputvtkDataObject that will be passed through after all iterations.
OutputvtkDataObject Shallow copy of the input

Online examples:

  • Cinema IO example
  • Contour Tree Alignment example
  • Nested Tracking from Overlap example
