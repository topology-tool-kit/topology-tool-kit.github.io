Detailed Description

TTK processing package for scalar field smoothing.

Author
Julien Tierny julie.nosp@m.n.ti.nosp@m.erny@.nosp@m.lip6.nosp@m..fr
Date
October 2014.

This class is a dummy example for the development of TTK classes. It smooths an input scalar field by averaging the scalar values on the link of each vertex.

Parameters
dataTypeData type of the input scalar field (char, float, etc.).
See also
ttkScalarFieldSmoother.cpp for a usage example.

Online examples:

  • 1-Manifold Learning example
  • 2-Manifold Learning example
  • Dragon example
  • Harmonic Skeleton example
  • Interaction sites example
  • Merge Tree Principal Geodesic Analysis example
  • Merge Tree Wasserstein Auto-Encoder example
  • Morse Molecule example
  • Morse-Smale segmentation example
  • MPI example
  • Persistence Diagram Principal Geodesic Analysis example
  • Topological Optimization DarkSky
  • Topological Optimization for Pegasus Genus Repair example
