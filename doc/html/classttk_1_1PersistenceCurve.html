Detailed Description

TTK processing package for the computation of persistence curves.

Author
Guillaume Favelier guill.nosp@m.aume.nosp@m..fave.nosp@m.lier.nosp@m.@lip6.nosp@m..fr
Julien Tierny julie.nosp@m.n.ti.nosp@m.erny@.nosp@m.lip6.nosp@m..fr
Pierre Guillou pierr.nosp@m.e.gu.nosp@m.illou.nosp@m.@lip.nosp@m.6.fr
Date
September 2016.

This package takes a ttk::DiagramType as input and computes the number of pairs as a function of persistence (i.e. the number of pairs whose persistence is higher than a threshold).

These curves provide useful visual clues in order to fine-tune persistence simplification thresholds.

See also
ttkPersistenceCurve.cpp for a usage example.

Online examples:

  • BuiltInExample1
  • Dragon example
  • Interaction sites
  • Morse Persistence example
